Ce programme en langage C, intitulé E-Cars, gère un système de location de voitures avec des fonctionnalités pour les membres de l'équipe et les clients  afin de faciliter la gestion des véhicules et des locations.
Si vous êtes un membre de l'équipe, Vous pouvez utiliser ce programme pour :
Ajoutez, supprimez, affichez des détails, modifiez la disponibilité, modifiez l'état de fonctionnement, affichez toutes les voitures, affichez l'historique de location, sauvegardez et quittez.
Si vous êtes un client, Vous pouvez utiliser ce programme pour :
Louez une voiture, réclamez et retournez une voiture, affichez toutes les voitures, affichez la description d'une voiture, sauvegardez et quittez.
